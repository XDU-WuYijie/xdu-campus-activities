# 项目协作约定

## 文档入口
- [功能模块设计](./功能模块设计.md)
- [工程进展](./工程进展.md)

## 更新规则
1. 每完成一个功能设计，必须同步更新 [功能模块设计](./功能模块设计.md)。
2. 每完成一个功能设计或实现里程碑，必须同步更新 [工程进展](./工程进展.md)。
3. 设计文档记录模块目标、接口、数据模型、页面入口和边界约束。
4. 进展文档记录当前状态、已完成项、待办项、阻塞项和下一步。

## 当前项目方向
- 项目正在从“商铺/博客/优惠券”业务迁移为“校园大型活动报名与签到平台”。
- 第一阶段主线是：活动门户、活动详情、发起活动、报名、报名名单、签到码签到。
