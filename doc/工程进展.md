## 当前阻塞
- 新活动表尚未导入现有数据库，运行时会缺少 `tb_activity` 和 `tb_activity_registration`
- 新签到凭证表尚未导入现有数据库，运行时会缺少 `tb_activity_voucher` 和 `tb_activity_check_in_record`
- 用户资料扩展表字段未导入时，运行时会缺少 `college/grade/mentor` 列
- 用户角色字段未导入时，运行时会缺少 `role_type` 列
- 活动图片字段未导入时，运行时会缺少 `images` 列
- 报名异步状态字段未导入时，运行时会缺少 `request_id/fail_reason/confirm_time` 列
- RBAC 表未导入时，系统会回退到 `role_type` 推导权限，无法验证正式角色绑定、权限回收和多角色场景
- `username` 与 `organizer_apply` 未导入时，管理员账号登录、主办方申请和审核流程无法运行
- 通知表未导入时，通知接口、业务通知落库和通知中心页面无法运行
- 未配置 RocketMQ 或 broker 未启动时，报名请求无法进入异步确认链路
- 未配置 RocketMQ 或 broker 未启动时，通知事件会走本地补偿，但无法验证异步消费链路
- 未启动 Elasticsearch 时，活动大厅会降级到 MySQL 查询，无法验证 ES 搜索相关性、聚合与同步链路
- Spring Boot 3.5.x / JDK 17 升级后，仍需验证 RocketMQ starter、ES 高阶客户端和 MyBatis-Plus 在本地环境的完整兼容性
